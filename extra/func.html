Дополнительные задания для изучения Функций

  1. Создайте функцию, которая возвращает только отрицательные числа

    Задание

  2. Создайте функцию, которая принимает два целых положительных числа (a, b) и возвращает список чисел от a до b включительно

    Задание

  3. Создайте функцию, которая определяет делится ли число x на y и z без остатка или нет

    Задание

  4. Создайте функцию, которая возвращает среднее значение суммы чисел в списке. Если список чисел будет пустой, тогда нужно вернуть 0

    Задание

  5. Создайте функцию, которая принимает два значения - число (repeat) и строку (string). А возвращает строку повторенную число раз.

    Задание

  6. Создайте функцию, которая возвращает наименьшее значение числа из списка

    Задание

  7. Создайте функцию, которая конвертирует булев тип в строку (True -> Yes, False -> No)

    Задание

  8. Создайте функцию, которая складывает значения двух списков

    Задание

  9. Создайте функцию, которая принимает строку обозначающую текущий цвет светофора и возвращающую следующий идущий за ним цвет. green -> yellow -> red -> green -> ...

    Задание

  10. Создайте функцию, которая преобразует год в век. 1900 -> 19 век

    Задание

  11. Создайте функцию, которая принимает 3 значения для создания последовательностей.
    start, stop, step
    И возвращает сумму чисел этой последовательности где финальное число используется включительно. Если последовательность чисел невозможно создать, тогда нужно вернуть 0.
    Например: 2, 6, 2 -> 2 + 4 + 6 = 12

    Задание

  12. Создайте функцию, которая отфильтровывает слова, используя специальный список из слов

    Задание

  13. Создайте функцию, которая возвращает строку с набором знаков (+) отображающих квадратную форму
    Например: 3 ->

    Шифр для декодирования, это позиция символов в английском алфавите, который написан наоборот

    Например: a -> z, x -> c
