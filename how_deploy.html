Инструкция как выложить сайт в интернет

  1. Нажимаем "New repository"

  2. В "Repository name" пишем Имя_логина.github.io

  3. Нажимаем "Create repository"

  4. Добавляем файлы в репозиторий. Для этого нажимаем на "uploading an existing file"

  5. Перетаскиваем index.html или выбираем его через "choose your files"

  6. Нажимаем на "Commit changes"

  7. Проверяем добавился ли файл в репозиторий

Как добавить новые файлы

  1. В основном репозитории нажимаем на Add file -> Upload files

Как загрузить страничку домой

  1. В основном репозитории нажимаем на Code -> Download ZIP
